# Skill: Screenshot

## Quando usar
Use esta skill quando o usuário pedir para capturar, tirar ou enviar uma imagem da tela atual.

Exemplos de frases que ativam esta skill:
- "tira um print"
- "me manda um screenshot"
- "captura a tela"
- "como está a tela agora"

## Como usar
Esta é uma ação nativa do bot. Não use ferramentas MCP para isso.
O bot já possui a função `capture_screenshot()` que captura e envia a imagem diretamente pelo Telegram.

## Nível de risco
read — não modifica nada no sistema.
